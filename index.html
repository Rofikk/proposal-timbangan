Define

Masalah & tujuan perbaikan

AS-IS (Saat Ini)

  • Timbangan dicatat di kertas
  • Handoff kertas dari petugas timbang ke admin
  • Verifikasi manual, keputusan tidak selalu terdokumentasi
  • Total transaksi harian dihitung manual
  • Supervisor rekap transaksi dan distribusi gudang pusat

Target (To-Be)

  • Auto-weighing RS232, minim salah tulis
  • Workflow digital end-to-end
  • Verifikasi bisa membatalkan dengan alasan wajib
  • Rekap harian otomatis + dashboard
  • Integrasi stok & distribusi gudang pusat

Measure

Indikator problem yang terlihat di lapangan
Error tulis Delay Rework Audit gap
Temuan dari Google Docs di sini: • Manual, rawan salah tulis; Kertas hilang/rusak; Tidak real-time, tidak auditable • Ketergantungan fisik; Potensi manipulasi angka; Proses lambat • Verifikasi informal; Tidak ada bukti keputusan; Sulit audit jika ada selisih • Hitung manual; Rawan salah total; Tidak real-time • Rekap manual; Data tersebar; Sulit analisa • Tidak sinkron antara timbang & Gudang; Selisih berat sulit ditelusuri

Analyse

Akar penyebab utama

Root Cause

  • Data timbang tidak ditangkap dari sumber (RS232 belum dimanfaatkan)
  • Workflow belum terdigitalisasi (handoff manual)
  • Verifikasi tanpa audit trail kuat
  • Rekap & total harian manual
  • Rantai timbang → gudang pusat tidak terkunci (traceability lemah)

Visual: Rantai Penyebab

Improve

Solusi yang diusulkan

TO-BE Flow

Catatan operasional: Petugas bisa timbang lebih dari satu jenis barang dalam 1 transaksi. Netto tetap bisa input manual sesuai SOP.

Control

Menjaga perbaikan tetap berjalan

Kontrol Proses

  • Role-based access: Petugas / Admin / Supervisor
  • Alasan wajib untuk cancel/koreksi
  • Status transaksi terkunci per tahap

Kontrol Audit & Data

  • Raw RS232 tersimpan
  • Log perubahan (who-when-what)
  • Backup & export rutin

Paket Penawaran

Budget cap sesuai arahan: BASIC 50jt, PRO 80jt, ENTERPRISE 140jt

BASIC

Rp 50.000.000 / project (max)

Mulai paperless & menangkap data timbang dari RS232.

Start
  • Auto-Weighing RS232 (raw + angka)
  • Timbang multi-item
  • Workflow dasar (handoff digital)
  • Master kode barang

PRO

Rp 80.000.000 / project (max)

Operasional harian: verifikasi kuat + auto-calc + rekap.

Best Value
  • Semua fitur BASIC
  • Verifikasi admin (bisa batal + alasan)
  • Auto-calculation (netto × harga/kg → total)
  • Rekap harian + export laporan
  • Audit trail transaksi

ENTERPRISE

Rp 140.000.000 / project (max)

Kontrol penuh: dashboard supervisor, gudang pusat, anomali.

Full Control
  • Semua fitur PRO
  • Dashboard supervisor & KPI
  • Gudang pusat (stok & distribusi / batch)
  • Deteksi selisih & anomali
  • SOP + training + UAT

RAB Ringkas per Paket

Disusun konsisten dengan scope paket
Paket Komponen Biaya Estimasi Catatan
BASIC Analisis + desain workflow dasar, modul timbang multi-item, integrasi RS232 (bridge reader), database & export dasar, dokumentasi. ≤ Rp 50.000.000 Fokus: auto-weighing + handoff digital.
PRO Semua BASIC + modul verifikasi (cancel + reason), auto-calculation, rekap harian, audit trail, laporan PDF/Excel. ≤ Rp 80.000.000 Fokus: percepat timbang→bayar & audit-ready.
ENTERPRISE Semua PRO + dashboard supervisor & KPI, modul gudang pusat (stok/distribusi/batch), deteksi anomali/selisih, SOP+training+UAT. ≤ Rp 140.000.000 Fokus: kontrol end-to-end termasuk gudang pusat.

Opsional (jika klien minta terpisah): perangkat PC timbangan, kabel/USB-Serial adapter, UPS, dan biaya hosting tahunan.
